## 1. 系统简介

本系统基于 YOLOv11 深度学习模型，实现对工业传送带图像中**螺母缺失状态**的自动检测。

### 检测类别

| 类别 | 说明 | 颜色标识 |
|---|---|---|
| **Nut_Missing** | 孔位存在但螺母缺失 | 橙色 |
| **Empty_Hole** | 空孔（未安装任何零件） | 蓝色 |

## 2. 环境配置

### 2.1 基础环境

- **Python**: 3.8 或更高
- **操作系统**: Windows 10+ / Linux / macOS
- **GPU**: 推荐使用 NVIDIA GPU（CUDA 加速）

### 2.2 安装依赖

```bash
pip install -r requirements.txt
```

> 💡 建议使用 conda 或 venv 创建独立的 Python 环境

### 2.3 GPU 配置（可选）

使用 GPU 推理可大幅提升速度：

```bash
# 检查 PyTorch GPU 支持
python -c "import torch; print(torch.cuda.is_available())"
```

## 3. 数据集准备

### 3.1 目录结构

```
datasets/missing/
├── images/
│   ├── train/      ← 训练图像（.jpg/.png）
│   ├── val/        ← 验证图像
│   └── test/       ← 测试图像
└── labels/
    ├── train/      ← YOLO TXT 标注（与图像一一对应）
    ├── val/
    └── test/
```

### 3.2 标注格式（YOLO TXT）

每张图像对应一个同名的 `.txt` 文件，格式如下：

```
<class_id> <x_center> <y_center> <width> <height>
```

- 所有坐标均为**归一化值**（0~1）
- `class_id`: 0 = Nut_Missing, 1 = Empty_Hole

**示例** (`001.jpg.txt`):
```
0 0.471 0.121 0.197 0.158
0 0.727 0.211 0.171 0.231
```

### 3.3 配置数据路径

编辑 `datasets/missing.yaml`，将 `path` 改为数据集实际路径：

```yaml
path: /你的/实际/路径/datasets/missing
```

## 4. 模型训练

### 4.1 快速开始

```bash
cd models
python train.py
```

### 4.2 训练参数说明

| 参数 | 默认值 | 说明 |
|---|---|---|
| `epochs` | 100 | 训练轮次（数据量小可增大） |
| `imgsz` | 640 | 输入分辨率（越大越准但越慢） |
| `batch` | -1 | 自动选择最优 batch |
| `cache` | "ram" | 缓存到内存加速（内存够用时开启） |

### 4.3 训练输出

训练完成后：
- 权重文件：`runs/detect/train*/weights/best.pt`
- 训练曲线：`results.png`
- 混淆矩阵：`confusion_matrix.png`

### 4.4 更换基础模型

编辑 `models/train.py`，将 `yolo11n.pt` 改为：

```python
model = YOLO("yolo11s.pt")   # 小模型（更快）
model = YOLO("yolo11m.pt")   # 中模型（平衡）
model = YOLO("yolo11l.pt")   # 大模型（更准）
```

## 5. GUI 使用指南

### 5.1 启动

```bash
cd gui
python app.py
```

### 5.2 加载模型

首次使用需加载训练好的 `.pt` 权重文件：
- 点击右下角 **「加载模型」** 按钮
- 选择 `best.pt` 或 `last.pt`

> 💡 也可以将权重文件放到 `gui/weights/best.pt`，系统会自动加载

### 5.3 功能按钮

| 按钮 | 功能 |
|---|---|
| 选择文件夹 | 批量导入文件夹内所有图片/视频 |
| 选择图片 | 单张图片检测 |
| 选择视频 | 视频文件检测（支持播放控制） |
| 停止播放 | 暂停视频播放 |
| 保存结果 | 将当前标注画面保存为图片 |
| 一键刷新 | 清空所有内容，重新开始 |

### 5.4 推理参数调整

编辑 `gui/app.py` 顶部的配置区：

```python
DEFAULT_CONFIDENCE = 0.3   # 置信度阈值（越大越严格）
DEFAULT_IOU = 0.45         # NMS IoU 阈值
DEFAULT_IMGSZ = 416        # 推理分辨率（越小越快）
```

## 6. 常见问题

### Q1: 训练时提示 `CUDA out of memory`

**解决**：减小 `batch` 和 `imgsz`

```python
model.train(batch=8, imgsz=416)
```

### Q2: 模型加载失败

**解决**：
1. 确认 `.pt` 文件路径正确
2. 确认 ultralytics 版本兼容（`pip install --upgrade ultralytics`）

### Q3: GUI 显示模糊（Windows）

**解决**：本系统已内置 DPI 自适应，若仍模糊可在桌面显示设置中调整缩放

### Q4: 视频播放卡顿

**解决**：
1. 使用 GPU 推理
2. 调低 `DEFAULT_IMGSZ`（如改为 320）
3. 增加 `skip_frame` 跳帧值（编辑 `gui/app.py`）
